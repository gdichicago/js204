Implementation

Changing parent methods and properties, or creating new ones that the parent does not have access to.

Using our Paperback (Book) from earlier:

// A new method on this object
PaperBack.prototype.burn = function() {
  console.log("Omg, you burnt all " + this.numPages + " pages");
  this.numPages = 0;
}

// Instantiating a new object
var paperback = new PaperBack("1984", "George Orwell", 250, "cover.jpg");
paperback.read();
paperback.burn();

Paperback has a method burn() that its parent object (Book) does not.
