Namespace in JS

Here is an example of a global namespace (Bird):

var Bird = Bird || {};

Bird.eat = function() {
  alert("Yummy yummy treats in my tummy tummy!");
};
Bird.eat();

And a subnamespace (say):

Bird.say = {};
Bird.say = {
  hello: function(name) {
    alert("Hello " + name + "!");
  }
};
Bird.say.hello("Liz");
