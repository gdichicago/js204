Inheritance

Access to the properties and methods created by the parent object(s).

Using our Book from earlier:

// The constructor function
function Paperback(title, author, numPages, cover) {
  Book.call(this, title, author, numPages);
  this.cover = cover;
}
// Extending the Book object
Paperback.prototype = Object.create(Book.prototype);

Because it inherits from Book, Paperback has the properties and methods available to book as well.
