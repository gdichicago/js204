Objects in OOP

Objects in object-oriented programming (OOP) have additional characteristics. Objects:

  • Are instances of a particular class
  • Are a combination of variables, functions, and data structures
  • Are primarily used to create modular, reusable pieces of code
  • Have a state (the data) and behavior (the code)
  • Send messages to each other about their state and behavior
  • Correspond to things found in the real world
  • Have hierarchies that can be used to group things together to share logic
